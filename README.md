# Ice Cream Shop - Login, Sign In, and Sign Up Pages

## Project Overview
Welcome to the Ice Cream Shop web application project. This project includes a fully functional login system with sign-in and sign-up forms.

## Features
- **User Authentication**: Secure login and registration.
- **Responsive Design**: Works on all devices.
- **Clean UI**: User-friendly interface.

## Pages
### Login Page
- **URL**: `/login`
- **Description**: This page allows users to log in using their email and password.
- **Example**:
  ```html
  <form action="/login" method="POST">
      <input type="email" placeholder="Email" required>
      <input type="password" placeholder="Password" required>
      <button type="submit">Login</button>
      <a href="#" id="forgotPassword">Forgot Password?</a>
  </form>
